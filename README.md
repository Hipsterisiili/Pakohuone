# Pakohuone
Kurssilla Tiralabra k2020 tehtävä harjoitustyö.
Pakohuone on sovellus, joka etsii tehokkaasti lyhyimpiä reittejä labyrintistä, joka sisältää avaimia ja ovia, joita avaimet avaavat.
